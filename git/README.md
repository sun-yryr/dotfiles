# Git関連の設定

このディレクトリには、Gitおよび関連ツールの設定ファイルが含まれています。

## 含まれるファイル

- `ghr/config.toml`: GitHubリポジトリ管理ツール「ghr」の設定ファイル

## 使用方法

インストールスクリプト（`install.sh`）を実行すると、これらの設定ファイルが適切な場所にシンボリックリンクされます。

## ghrについて

ghrは、GitHubリポジトリをローカルに一括でクローンしたり管理したりするためのツールです。主な機能は以下の通りです：

- リポジトリの一覧表示
- リポジトリのクローン
- リポジトリの更新

詳細については、[ghrのGitHubリポジトリ](https://github.com/siketyan/ghr)を参照してください。
